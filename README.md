# NEV1
Испытание рационального, поведенческого и риск-чувствительного Q-Learning агентов в среде из Лабораторной №1. Рассматривается задача управления автономным дроном в двумерном пространстве X = [0, 20] × [0, 10]. Целью дрона является достижение станции быстрой зарядки при максимизации накопленного вознаграждения с учетом стохастического ветра, энергопотребления и статичных препятствий.
